Tutorial #1 : How to start Dofus 1.29


Some information about the game

Dofus is a MMORPG french game released in 2004 by Ankama. This game can be played either in free-to-play or pay-to-play mode. The game has been developed in Adobe Flash programming language. Since its release, the game has been updated several times and Ankama Games keeps releasing new content every month. However, this tutorial will focus on a previous version of the game, the 1.29 version. To contextualize, Ankama Games evolved the game architecture by upgrading the game architecture from 1.29 to 2.0 version. This feature led to major changes in term of graphic aspect and features as it is shown in the following images (Dofus 1.29 on the left and 2.0 on the right).



All the servers were migrated to that new version, well all of them except two servers. Since some users in Latin America had poor PC performances back in 2009, Ankama Games decided to keep two active servers for that community. And guess what, the servers are still accessible in 2020! This version can be enjoyable both for young and elderly. This tutorial will provide a full guide on how to start the game Dofus. This whole tutorial will assume the user to have a subbed account (which can easily be done in the Dofus website)

Steps to start an adventure in Dofus

  1. Download Dofus 1.29
  2. Create an account
  3. Create your character
    • Choose your server
    • Choose your class
    • Choose your appearance and name
  4. Explore the first world : Incarnam



1. Download Dofus 1.29

First of all, to play the game, the user must download the Ankama Launcher. This tool is similar to the Epic Launcher or the Origin Launcher. It allows the user to download all the games belonging to Ankama Games, including Dofus 1.29. To download the Ankama Launcher, please click here. You can choose your OS by clicking Download a Different Version highlighted in the following image. Please note that the Ankama Launcher is not guaranteed to run on all Linux distributions.



After downloading the Ankama Launcher, in the section "games", select the second game with the logo and click download. The download progression should be indicated on the bottom-right corner. Please note that other activities such streaming, downloading or suspicious website browsing should be avoided during the downloading if the internet connexion speed is low.



2. Create an account

For this part, the Dofus website will also be useful. Click here to create an account. It is possible to create an Ankama account from a Steam account. The account name selection must be done carefully since it is not possible to modify the username.After filling all the requirements, click Complete Registration.

3. Create your character

To create a character in the game Dofus 1.29, the user has to choose the characteristics of his character. Those characteristics are the character's class, appearance and name and server.

Choose your server

Dofus 1.29 had from 2009 to 2019 only two available servers. However, there was a huge nostalgic community in game that led Ankama Games to develop more servers to match the demand. Currently, there are 11 available servers : Ayuto, Clustus, Bilby, BiIssering, Eratz, Henual. Mabur, Arty. Droupik, Hogmeiser and Algathe. Among all these servers, Eratz and Henual are the oldest server. Depending on the user's interests, some servers can be better. For example, some servers are more populated than others. This can be an advantage if a player wants to find more teammates to play with or a neutral aspect if the player does not want to play with other players. It is possible to know if a server is very populated. To choose you server, launch Dofus 1.29 via the Ankama Launcher, log in and click Create a character. After, click I want to select my server and check Display all servers.

Each server shows its level of population, from Recommanded to High. High population servers are recommended for beginners. Then, click on the chosen server and apply Select.


Choose your class

Dofus is an MMORPG. Thus, players have to cooperate together to achieve and unlock new contents. That is why Dofus have a class system. It allows players to have different specializations. Each class is unique and allows to add synergy to an existing group. Here is a short table about the 12 available classes and their utilities.

Class Description
Sadida Nature's ally, he uses poisons to deal damages. Summon puppies and trees to limit the enemy's movements. Can also be an important damage dealer. This class is recommended for players who prefer to play alone.
Osamodas This class relies on summoning creatures. Is also a good support who can provide boosting, healing and protection to allies. Matches well with big teams but can also be played in solo.
Enutrof This class is the best for dropping items, resources and money. Can also be a good damage dealer and has a good ability to survive in tight spots
Sram This class can turn invisible and allows teammates to be invisible. Can also set invisible traps and make a copy of itself. Is also a good damage dealer. This class is recommended for lonely people who do not like social interactions
Xelor Master of time, this class allows its user to limit opponents' actions. Has also a huge mobility capacity. Is useful in teams because it can gives Ability Points to its teammates. Not recommended for beginners
Pandawa This class requires to be drunk (in game) to attack. This drunk mode gives him less movement capacity but gives him more protection. Can be a important AOE (area of effect) damage dealer and an important control manager
Ecaflip This class heavily relies on luck. Some attacks could deal a lot of damage or even heal your opponents depending on your luck. This class is recommended for impulsive gamblers
Eniripsa This class is the healer of the game. Half of its abilities are designed to heal allies. Can also shield allies. Very useful in teams and are generally appreciated by other players
Iop This class is the main damage dealer of the game. Has an important mobility capacity and can deliver a lot of damage in few attacks. Is recommended for beginners
Cra This class can attack from afar. Usually used to play in long distances. Is also very useful for its AOE capacity. Recommended for solo players.
Feca This class is the main protector of Dofus. It can shield teammates and allow them to live longer. Has also AOE attacks. This class fits both solo and multiplayer playstyles.
Sacrieur This class is the best tank among the 12 classes. Can tank damages for the team and fight head-to-head with opponents. Is very useful in teamfights, but also revelant in solo gameplay.

Please be aware that the class selection is a permanent decision and the player cannot change his class after the character creation. To learn more about classes and their abilities , a user can acces the list of every class' spells by clicking Class spells under the class' image


Choose your appearance and name

The player can customize his character appearance by selecting the name, the gender and the color of its character. The player can customize three colors : the skin, the hair and the clothes. For the name, its up to the player to choose it, as long as nobody in the past already took it. Finally, it is possible to choose between male and female for every character.



4. Explore the first world : Incarnam

Congratulations! You just created your first character in the game Dofus. But there is more. It is time to explore the world of the 12, but specifically the first world : Incarnam. This world is designed to teach basic concepts about the game. To start with, you can moove from map to map by clicking the yellow and green suns at each extremity of a map (please see image).



From the starting point, go left and engage a fight with the Incarnam scarecrow. To engage a fight, click on the scarecrow. You are now in fight mode! But do not panic, it is very easy. To understand how a fight works, here is a screenshot during a fight.



  1. Ability points : players have to pay a certain amount of AP to cast a spell. This number reinitialize at the end of each turn
  2. Movement points : players have to pay a certain amount of MP to move. This number reinitialize at the end of each turn
  3. Health bar : if this bar reaches 0, the player has lost the fight
  4. Spells : Each box is a spell. Players must use spells to kill enemies
  5. Timeline : Indicates the order of each players and enemies' turn

To kill the scarecrow, use abilities on the enemy. You have now killed your first enemy, congratulations! With this tutorial, you can now visit Incarnam and fight enemies as long as you want! Here is a map of Incarnam that could be useful. To acces that map, click on the compass at any time.
